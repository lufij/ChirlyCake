# Imágenes para Formulario Público

Esta carpeta debe contener las imágenes de los tipos de cobertura:

## Archivos necesarios:

1. **turron.jpg** - Foto de pastel con cobertura de turrón/merengue
2. **betun.jpg** - Foto de pastel con cobertura de betún/crema
3. **fondant.jpg** - Foto de pastel con cobertura de fondant

## Cómo obtenerlas:

### Opción 1: Descargar del entorno actual
Si estás trabajando en el entorno de Figma Make, las imágenes actuales son:
- Turrón: Pastel con merengue de colores pastel
- Betún: Pastel rosa con macarons y "Feliz Cumpleaños"
- Fondant: Pastel con fondant y arcoíris

### Opción 2: Usar tus propias fotos
1. Toma fotos de pasteles reales de tu pastelería
2. Recórtalas a formato cuadrado (recomendado: 800x800px)
3. Guárdalas con los nombres exactos: turron.jpg, betun.jpg, fondant.jpg

### Opción 3: Usar fotos de stock
Puedes usar fotos de Unsplash o Pexels:
- Busca: "buttercream cake"
- Busca: "meringue cake"
- Busca: "fondant cake"

## Formato recomendado:
- Formato: JPG o PNG
- Tamaño: 800x800px o similar (cuadrado)
- Peso: Máximo 500KB por imagen
- Calidad: Alta, pero optimizada para web

## Una vez agregadas:

Verifica que los nombres sean exactamente:
- `turron.jpg` (o .png)
- `betun.jpg` (o .png)
- `fondant.jpg` (o .png)

Y actualiza las extensiones en `PublicOrderForm.tsx` si usas PNG en vez de JPG.
