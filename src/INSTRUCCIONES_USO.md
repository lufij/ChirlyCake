# Instrucciones de Uso - Sistema de Pastelería

## Sistema de Autenticación Simplificado

### Registro de Nueva Cuenta

**Cualquier persona puede registrarse:**
1. Abre la aplicación en tu navegador
2. Haz clic en la pestaña "Registrarse"
3. Completa el formulario:
   - **Nombre**: Tu nombre
   - **Apellido**: Tu apellido
   - **Número de Celular**: Será tu contraseña (ej: 71234567)
4. Haz clic en "Crear Cuenta"
5. ¡Listo! Serás redirigido automáticamente al sistema

**Notas importantes:**
- **Primer usuario**: Si eres el primer usuario en registrarte, automáticamente serás **Propietario**
- **Código de propietario**: Si necesitas crear una cuenta de propietario adicional, haz clic en "¿Eres propietario? Código especial" e ingresa el código: `PROPIETARIO2024`
- **Usuarios regulares**: Las cuentas normales tienen rol de "Vendedor". El propietario puede cambiar roles desde "Gestión de Usuarios"

### Para Usuarios (Empleados)

**Inicio de Sesión:**
1. Abre la aplicación en tu navegador
2. Ingresa tu número de celular (ejemplo: 71234567)
3. Haz clic en "Iniciar sesión"
4. Serás redirigido al panel principal según tu rol

**Nota:** No hay campo de contraseña visible. El sistema usa tu número de celular como contraseña de forma automática.

### Para Propietarios

**Creación de Nuevos Usuarios:**
1. Inicia sesión con tu número de celular
2. Ve a la pestaña "Usuarios"
3. Haz clic en "Nuevo Usuario"
4. Completa el formulario:
   - **Nombre**: Nombre del empleado
   - **Apellido**: Apellido del empleado
   - **Número de Celular**: Este número será usado para iniciar sesión
   - **Rol**: Selecciona el rol apropiado
5. Haz clic en "Crear Usuario"

**El nuevo empleado podrá iniciar sesión usando su número de celular.**

## Roles y Permisos

### Vendedor
- ✅ Gestionar pedidos (crear, editar, ver)
- ✅ Gestionar clientes
- ✅ Ver calendario de entregas
- ❌ No tiene acceso a finanzas
- ❌ No puede crear/eliminar usuarios

### Administrador
- ✅ Todo lo del Vendedor
- ✅ Ver reportes
- ✅ Gestionar usuarios (crear, editar, eliminar)
- ❌ **No tiene acceso a finanzas** (cambio reciente)

### Propietario
- ✅ Todo lo del Administrador
- ✅ **Acceso completo a finanzas**
- ✅ Ver transacciones
- ✅ Registrar ingresos/egresos
- ✅ Control total del sistema

## Configuración Inicial

### Primera Vez que Usas la Aplicación

Cuando abres la aplicación por primera vez y no hay usuarios registrados:
1. Verás una pantalla de "Configuración Inicial"
2. Crea tu cuenta de **Propietario** con:
   - Nombre
   - Apellido
   - Número de celular
3. Esta cuenta tendrá acceso completo al sistema

### Después de la Configuración Inicial

- **Opción 1**: Los empleados pueden auto-registrarse usando la pestaña "Registrarse" (tendrán rol de Vendedor)
- **Opción 2**: El propietario puede crear usuarios desde la sección "Gestión de Usuarios" y asignar roles específicos
- **Opción 3**: Para crear propietarios adicionales, usa el código especial `PROPIETARIO2024` durante el registro

## Notas Importantes

- **Seguridad**: El número de celular funciona como contraseña. Asegúrate de que cada empleado use un número único.
- **Formato del número**: Ingresa el número sin espacios ni guiones (ejemplo: 71234567, no 7-123-4567)
- **Números únicos**: Cada número de celular solo puede registrarse una vez. Si intentas registrar un número ya existente, recibirás un error.
- **Recuperación de acceso**: Si un empleado olvida su número o pierde acceso, contacta al propietario para verificar su información.
- **Cambio de rol**: Solo el propietario puede cambiar el rol de un usuario desde la sección "Gestión de Usuarios".

## Soporte

Para problemas técnicos o dudas sobre el sistema, contacta al administrador del sistema.
